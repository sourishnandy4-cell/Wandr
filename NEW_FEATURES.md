# 🎉 New Features Added!

## ✅ What's New

### 1. **Username Setup** (One-time)
- First time you open the app, you'll see a welcome screen
- Enter your name once
- It's saved forever in your browser
- Shows up in header and sidebar

### 2. **Add Friends Feature**
- Green "Friends" button in the dashboard
- Add trip members by name
- See how many friends are on the trip
- Copy shareable link to invite others
- Remove friends if needed

### 3. **No More "Sarah J."**
- Your actual username shows everywhere
- Header shows your name
- Sidebar shows your initials
- Personalized experience!

---

## 🎯 How It Works

### First Time Use:
1. Open app → See "Welcome to Wandr!" screen
2. Enter your name → Click "Continue"
3. Create your first trip
4. Start adding activities and expenses!

### Adding Friends:
1. Click green **"Friends (0)"** button
2. Type friend's name → Click "Add"
3. Copy the share link
4. Send link to friends
5. They can view the same trip!

### Sharing Trips:
- Each trip gets a unique link
- Friends can open the link to see trip details
- Everyone sees the same itinerary and expenses
- Perfect for group travel planning!
